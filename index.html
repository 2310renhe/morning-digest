microgpt (http://karpathy.github.io/2026/02/12/microgpt/)

Extract ALL key points from the source. Cover every major argument, data point, and conclusion. Use bullet points for clarity:

* The microgpt project is a 200-line Python script that trains and infers a GPT model with no dependencies.

* The script is a culmination of multiple projects (micrograd, makemore, nanogpt, etc.) and a decade-long obsession to simplify LLMs to their bare essentials.

* The script contains the full algorithmic content of what is needed: dataset of documents, tokenizer, autograd engine, a GPT-2-like neural network architecture, the Adam optimizer, training loop, and inference loop.

* The dataset used is a list of 32,000 names, one per line, which is used to train the model to generate new, plausible-sounding names.

* The tokenizer is a simple character-level tokenizer that assigns a unique integer to each character in the dataset.

* The autograd engine is implemented from scratch using a single class called `Value`, which wraps a scalar number and tracks how it was computed.

* The `Value` class has a `backward` method that walks the computation graph in reverse topological order, applying the chain rule at each step.

* The model architecture is a stateless function that takes a token, a position, the parameters, and the cached keys/values from previous positions, and returns logits over what token the model thinks should come next in the sequence.

* The model uses a combination of attention and MLP blocks to process the input sequence.

* The attention block projects the current token into three vectors: a query, a key, and a value, and uses these vectors to compute attention weights and a weighted sum of the cached values.

* The MLP block is a two-layer feed-forward network that projects up to 4x the embedding dimension, applies ReLU, and projects back down.

* The model uses residual connections to add the output of each block back to its input.

* The final hidden state is projected to vocabulary size by `lm_head`, producing one logit per token in the vocabulary.

* The script is available on GitHub and can be run as a Google Colab notebook.

* The author has also created a video explaining the autograd engine and the `Value` class.

* The script has been simplified to its bare essentials, with no unnecessary dependencies or code.

* The model is trained using a simple Adam optimizer and a cross-entropy loss function.

* The script generates new, plausible-sounding names by sampling from the trained model.

* The generated names are printed to the console, along with their corresponding token IDs and positions in the sequence.

Resolution-Aware Perpetual Futures on Binary Prediction Markets: An Empirical Risk-Design Framework Using Polymarket Data(https://arxiv.org/abs/2605.10400)

  • Develops a resolution-aware risk-design framework (PIRAP) for perpetual futures
  • The framework has six components: index estimator, jump-aware tiered margin, leverage compression schedule, resolution-aware funding rule, multi-stage halt protocol, and eligibility framework
  • Empirical evaluation uses Polymarket's PMXT v2 archive for 2026-04-21 to 2026-04-27
  • Results are mixed, with five pre-registered floors: two stylized-fact floors pass, two welfare-side directional floors fail, and one passes
  • The paper concludes with structural recommendations and explicit non-deployable status

A Taxonomy of Event-Linked Perpetual Futures: Variant Designs Beyond the Single-Market Binary Case(https://arxiv.org/abs/2605.10428)

  • Develops a formal taxonomy of seven pure-form canonical variants beyond the probability-index perpetual
  • Variants include conditional probabilities, spreads, weighted baskets, derivatives on variance or entropy, contracts on liquidity, perpetual-on-expiring-event roll structures, and funding-only derivatives
  • Each variant has a precise payoff definition, inheritance map, design constraints, microstructure properties, and evaluability criteria
  • Notable findings include a candidate non-portability proposition for the conditional variant and requirements for multi-period jump-aware margin in the basket variant

Manipulation, Insider Information, and Regulation in Leveraged Event-Linked Markets(https://arxiv.org/abs/2605.10486)

  • Develops a theoretical framework for leverage, manipulation, and informed-trading in event-linked markets
  • Leverage changes manipulation incentives and interacts with informed-trading rents
  • A two-axis manipulation taxonomy distinguishes market-price manipulation from real-world outcome manipulation
  • Section 7 connects pre-emption and halt-protocol findings to three manipulation channels
  • The paper concludes with 14 recommendations for venue operators, regulatory bodies, and the research community

Estimation of an Order Book Dependent Hawkes Process for Large Datasets(https://arxiv.org/abs/2307.09077)

  • Presents a point process for event arrivals in high-frequency trading with order book-dependent intensity
  • Conditions for stationarity of the process are stated
  • An algorithm is presented to estimate the model even with billions of data points
  • Convergence of the algorithm is shown, and consistency results are established
  • A test statistic to assess out-of-sample performance is suggested
  • The methodology is applied to four stocks trading on the New York Stock Exchange (NYSE)

Hierarchical Reinforced Trader (HRT): A Bi-Level Approach for Optimizing Stock Selection and Execution(https://arxiv.org/abs/2410.14927)

  • Proposes a bi-level reinforcement learning framework for text-aware portfolio management
  • HRT separates trading into two coordinated decisions: a High-Level Controller (HLC) selects asset-level directions, and a Low-Level Controller (LLC) converts directions into feasible portfolio weight adjustments
  • Evaluation on an open stock-news benchmark with a fixed 89-stock Nasdaq universe shows HRT delivers the strongest learning-based return-risk-cost trade-off
  • The full model improves Sharpe ratio, reduces daily turnover, and remains robust under transaction-cost stress

The Invisible Handshake: Persistent Overpricing by Adaptive Market Agents(https://arxiv.org/abs/2510.15995)

  • Studies overpricing in a repeated game between a market maker and a market taker
  • Market prices evolve through endogenous price impact and exogenous shocks
  • A sufficient condition for decentralized learning to reach the overpricing region is provided
  • Projected stochastic gradient ascent satisfies this condition
  • A decomposition of the game into competitive and collaborative components shows how decentralized learning leads to persistent overpricing
  • The same structural incentives govern both myopic and farsighted objectives

The Engineering of Skew: A Path-Dependent Framework for Asymmetric Volatility Management (https://arxiv.org/abs/2605.09123)

  • The paper develops a path-dependent framework for asymmetric volatility management
  • Institutions experience risk through drawdowns, liquidity needs, spending rules, rebalance decisions, board oversight, and the interval between a prior high-water mark and full recovery
  • The arithmetic of recovery is nonlinear: after a drawdown of depth D, the required gain is R = 1 / (1 - D) - 1
  • Lower volatility can improve geometric compounding, but symmetric de-risking can also impair recovery if it sacrifices too much upside participation
  • Skew engineering is defined as the portfolio construction discipline of reducing harmful downside participation more than productive upside participation
  • The Recovery-Efficiency Protocol links drawdown depth, time underwater, recovery burden reduction, and rebound participation into an allocator-facing reporting discipline
  • Machine learning and AI methods are framed as tools for conditional estimation, regime mapping, robustness testing, and model-risk governance

Beyond ESG Scores: Learning Dynamic Constraints for Sequential Portfolio Optimization (https://arxiv.org/abs/2605.09310)

  • The paper proposes a method for imposing ESG constraints without modifying the financial policy's observation or reward
  • A Multimodal Action-Conditioned Constraint Field (MACF) learns mechanism-specific ESG costs from point-in-time multimodal evidence and contemplated portfolio transitions
  • MACF-X is a family of optimizer-specific adapters that converts MACF costs and uncertainties into native constrained-optimization interfaces
  • The method reduces tail ESG budget pressure while maintaining competitive financial performance
  • Ablations show that this improvement depends on dynamic evidence inputs and three-head decomposition

Quantifying the Risk-Return Tradeoff in Forecasting (https://arxiv.org/abs/2605.09712)

  • The paper treats forecast loss differentials relative to a benchmark as a return series
  • It evaluates these returns using risk-adjusted performance measures from finance, including the Sharpe ratio, Sortino ratio, Omega ratio, and drawdown-based metrics
  • The Edge Ratio captures a model's propensity to deliver uniquely informative predictions relative to the forecasting frontier
  • The framework is applied to U.S. macroeconomic forecasting, comparing econometric benchmarks, machine learning models, a foundation model, and the Survey of Professional Forecasters
  • While it is often feasible to beat professional forecasters in terms of average accuracy, it is much harder to beat them on a risk-adjusted basis

Goal-based portfolio selection with mental accounting (https://arxiv.org/abs/2506.06654)

  • The paper presents a continuous-time portfolio selection framework that reflects goal-based investment principles and mental accounting behavior
  • An investor with multiple investment goals constructs separate portfolios, each corresponding to a specific goal, with penalties imposed on fund transfers between these goals
  • The value function is the unique constrained viscosity solution of a Hamilton-Jacobi-Bellman equation system
  • Numerical analysis reveals that the free boundaries exhibit complex shapes with bulges and notches, and the optimal strategy for one portfolio depends on the wealth level of another

Machine Learning Enhanced Multi-Factor Quantitative Trading: A Cross-Sectional Portfolio Optimization Approach with Bias Correction (https://arxiv.org/abs/2507.07107)

  • The paper proposes a method for multi-factor quantitative trading that resolves the issue of "upstream contamination" in rolling-window factor pipelines
  • A mask-first design constructs a Boolean tradability mask at data load time and threads it through every operator
  • The system adds a GPU-vectorised factor engine, an Adjusted-MSE loss, block-bootstrap GBM augmentation, and Markowitz-Ledoit-Wolf portfolio optimisation
  • On real A-share data, the system achieves an annualised Sharpe ratio of 1.63
  • Ablation shows that the mask contract is the single largest contributor to the system's performance

Large-Scale Asset Selection via Metric Dependence with Enriched High Frequency Information (https://arxiv.org/abs/2605.02326)

  • The paper proposes a method for large-scale asset selection that incorporates high frequency information as object-valued data
  • Each asset day observation is represented as a point-curve object combining daily return with an intraday risk state curve

Single-Period Portfolio Selection via Information Projection (https://arxiv.org/abs/2605.03184)

  • The paper studies the single-period portfolio selection problem under Constant Relative Risk-Aversion (CRRA) utility through the information-theoretic lens
  • The Certainty-Equivalent (CE) growth rate under CRRA utility can be decomposed into a portfolio-induced Rényi divergence term, a Rényi entropy term, and a log-partition term
  • CRRA portfolio selection is equivalent to a Rényi information-projection problem
  • The method uses a variational representation of Rényi divergence and obtains a Blahut-Arimoto-style alternating optimization
  • In the low risk-aversion regime, this method empirically requires fewer iterations than both direct CRRA utility optimization and Cover's method

A Market-Rule-Informed Neural Network for Efficient Imbalance Electricity Price Forecasting(https://arxiv.org/abs/2605.09061)

  • The paper proposes a market-rule-informed neural forecasting framework for imbalance electricity price forecasting.

Robust financial calibration: a Bayesian approach for neural SDEs(https://arxiv.org/abs/2409.06551)

  • The paper presents a Bayesian framework for the calibration of financial models using neural stochastic differential equations (neural SDEs).
  • The method is based on the specification of a prior distribution on the neural network weights and an adequately chosen likelihood function.
  • The resulting posterior distribution can be seen as a mixture of different classical neural SDE models yielding robust bounds on the implied volatility surface.
  • Both historical financial time series data and option price data are taken into consideration.
  • A Langevin-type algorithm is used for robust numerical optimization of the neural networks.

OrderFusion: Encoding Orderbook for End-to-End Probabilistic Intraday Electricity Price Forecasting(https://arxiv.org/abs/2502.06830)

  • The paper develops a new order fusion methodology for end-to-end and parameter-efficient probabilistic forecasting of intraday electricity prices.
  • The approach learns a interaction-aware representation of the buy-sell dynamics.
  • The methodology hierarchically estimates the quantiles with non-crossing constraints.
  • Extensive experiments on CID price indices across high- and low-liquidity European markets demonstrate consistent improvements over conventional baselines.
  • The methodology is available at: https://runyao-yu.github.io/OrderFusion/.

Systemic Risk in the European Insurance Sector(https://arxiv.org/abs/2505.02635)

  • The paper studies systemic-risk connectedness in the European insurance sector at three levels of granularity.
  • The analysis documents that insurers are an important component of systemic-risk connectedness, especially during stress episodes.
  • Aggregate insurer spillovers co-move with term spreads, sovereign spreads, and funding stress.
  • Firm-level insurer-to-bank spillovers vary with sovereign risk and domestic sovereign-bond home bias in a way consistent with a balance-sheet channel.
  • The analysis reveals substantial heterogeneity across subsectors and identifies a stable core of systemically central insurers in firm-level networks.

Analytic approximation for Bachelier option prices and applications(https://arxiv.org/abs/2605.02040)

  • The paper writes the price for out-of-the-money (OTM) and in-the-money (ITM) options as an expansion with respect to the moneyness.
  • The coefficients are related to the negative (non-integer) powers of the future mean volatility.
  • The expansion is used as a control variate to reduce the variance of Monte Carlo option prices in the correlated case.

Convergence in probability of numerical solutions of a highly non-linear delayed stochastic interest rate model(https://arxiv.org/abs/2510.04092)

  • The paper studies a delayed stochastic interest rate model with superlinearly growing coefficients.
  • Novel analytical tools are developed to investigate the properties of both the true solution and its truncated Euler-Maruyama (TEM) approximation.
  • The true solution converges in probability to the truncated EM solution as the step size approaches zero.
  • Numerical experiments validate the convergence results using an efficient Monte Carlo simulation framework for the valuation of relevant financial quantities.
