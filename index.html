microgpt (http://karpathy.github.io/2026/02/12/microgpt/)

Extract ALL key points from the source. Cover every major argument, data point, and conclusion. Use bullet points for clarity:

* The microgpt project is a 200-line Python script that trains and infers a GPT model with no dependencies.

* The script is a culmination of multiple projects (micrograd, makemore, nanogpt, etc.) and a decade-long obsession to simplify LLMs to their bare essentials.

* The script contains the full algorithmic content of what is needed: dataset of documents, tokenizer, autograd engine, a GPT-2-like neural network architecture, the Adam optimizer, training loop, and inference loop.

* The dataset used is a list of 32,000 names, one per line, which is used to train the model to generate new, plausible-sounding names.

* The tokenizer is a simple character-level tokenizer that assigns a unique integer to each character in the dataset.

* The autograd engine is implemented from scratch using a single class called `Value`, which wraps a scalar number and tracks how it was computed.

* The `Value` class has a `backward` method that walks the computation graph in reverse topological order, applying the chain rule at each step.

* The model architecture is a stateless function that takes a token, a position, the parameters, and the cached keys/values from previous positions, and returns logits over what token the model thinks should come next in the sequence.

* The model uses a combination of attention and MLP blocks to process the input sequence.

* The attention block projects the current token into three vectors: a query, a key, and a value, and uses these vectors to compute attention weights and a weighted sum of the cached values.

* The MLP block is a two-layer feed-forward network that projects up to 4x the embedding dimension, applies ReLU, and projects back down.

* The model uses residual connections to add the output of each block back to its input.

* The final hidden state is projected to vocabulary size by `lm_head`, producing one logit per token in the vocabulary.

* The script is available on GitHub and can be run as a Google Colab notebook.

* The author has also created a video explaining the autograd engine and the `Value` class.

* The script has been simplified to its bare essentials, with no unnecessary dependencies or code.

* The model is trained using a simple Adam optimizer and a cross-entropy loss function.

* The script generates new, plausible-sounding names by sampling from the trained model.

* The generated names are printed to the console, along with their corresponding token IDs and positions in the sequence.

The algorithm constructs a vector-formed mirror statistic for FDR control with the Gaussian Mirror algorithm and proposes a new kernel-based independence measure. CatNet performs robustly on different model settings with both simulated and real-world data.

Numerical methods for lambda quantiles: robust evaluation and portfolio optimisation (https://arxiv.org/abs/2605.06220)

This work presents efficient algorithms for computing lambda quantiles and demonstrates their application in portfolio optimisation. A robust algorithm, Λ-Newton-Bis, is developed to ensure global convergence. The algorithm's computational efficiency and practical relevance are demonstrated within a portfolio optimization framework.

A Geometry-Aware Residual Correction of Hagan's SABR Implied Volatility Formula (https://arxiv.org/abs/2605.06604)

This paper proposes a hybrid methodology to improve the approximation of SABR implied volatility by combining analytical structure with machine learning. The approach augments the neural-network input representation with geometric features derived from the stochastic differential equations of the SABR model. Numerical experiments show that the method improves accuracy and robustness compared to both analytical approximations and standard neural-network approaches.

Frustrated Dynamics of Distance Matrices (https://arxiv.org/abs/2605.05376)

The Frustrated Distance Matrix (FDM) model is introduced, a dynamic extension of the static distance-matrix ensemble. The model's entries are pairwise geodesic distances between N Brownian particles on the sphere evolving under quenched random pairwise couplings. The paper finds that the static BBS theory is preserved at every time, with the dynamics entering as a redistribution of spectral mass within that template.

Multi-Dimensional Behavioral Evaluation of Agentic Stock Prediction Systems Using LLM Judges with Closed-Loop Reinforcement Learning Feedback (https://arxiv.org/abs/2605.05739)

This work presents a behavioral evaluation framework for agentic stock prediction systems, addressing the gap in individual decision quality hidden by aggregate metrics. The framework uses an ensemble of large language model judges to score behavioral traces along six domain-specific dimensions. The framework is used to convert deficient per-dimension scores into a penalty term added to the Soft Actor-Critic reward, resulting in improved performance.

INEUS: Iterative Neural Solver for High-Dimensional PIDEs (https://arxiv.org/abs/2605.06281)

INEUS, a meshfree iterative neural solver for partial integro-differential equations (PIDEs), is introduced. The method replaces explicit evaluation of nonlocal jump integrals with single-jump sampling and reformulates PIDE solving as recursive regression problems. INEUS delivers accurate and scalable solutions for various high-dimensional linear and nonlinear examples.

SNAPO: Smooth Neural Adjoint Policy Optimization for Optimal Control via Differentiable Simulation (https://arxiv.org/abs/2605.06570)

SNAPO, a framework that embeds a neural policy inside a known, differentiable simulator, is introduced. SNAPO computes exact gradients of the objective with respect to all policy parameters and inputs in a single adjoint pass. The framework is demonstrated on three domains, including natural gas storage, pension fund asset-liability management, and pharmaceutical manufacturing.

Proactive Market Making and Liquidity Analysis for Everlasting Options in DeFi Ecosystems (https://arxiv.org/abs/2508.07068)

This paper analyzes markets for everlasting options using a dynamic proactive market maker. The behavior of funding fees and transaction costs across varying liquidity conditions is examined. The paper demonstrates that liquidity providers can achieve a net positive PnL by employing effective hedging strategies.

Analytic approximation for Bachelier option prices and applications (https://arxiv.org/abs/2605.02040)

An analytic approximation for Bachelier option prices is derived using classical Itô calculus and Taylor expansions. The price for out-of-the-money and in-the-money options is written as an expansion with respect to the moneyness. The approximation is used as a control variate to reduce the variance of Monte Carlo option prices.

What Can Go Wrong During Caplet Stripping (https://arxiv.org/abs/2605.05140)

This paper studies exact and near-exact extraction of caplet volatilities from market cap quotes and identifies sources of instability. Practical remedies, including continuous flat-linear and C1 flat-smooth kernels, are proposed to preserve bootstrap equivalence and enforce positivity.

On Utility Maximization under Multivariate Fake Stationary Affine Volterra Models (https://arxiv.org/abs/2603.11046)

This paper tackles Merton's portfolio optimization problem in a Volterra stochastic environment described by a multivariate fake stationary Volterra-Heston model. The problem is solved using a stochastic factor solution to a Riccati backward stochastic differential equation (BSDE). The resulting optimal strategies are derived in semi-closed form, depending on the solutions to time-dependent multivariate Riccati-Volterra equations.
